Q: JavaScript, CSS, HTML  有什麼區別?

JavaScript、CSS 和 HTML 都是用於網頁開發的語言，但是它們的功能和用途有所不同：

HTML (Hypertext Markup Language) 是用於創建網頁的標記語言，它定義了網頁的結構和內容，包括文字、圖像、超連結等等。HTML 是網頁的基礎，所有的網頁都必須使用 HTML 來建立。

CSS (Cascading Style Sheets) 是用於設計網頁樣式的語言，它可以控制網頁的佈局、字體、顏色、大小、邊框、背景等等。CSS 可以讓網頁更具美感和可讀性，提高網頁的使用體驗。

JavaScript 是一種編程語言，可以讓網頁具有互動性和動態效果。JavaScript 可以讓網頁對用戶的操作做出反應，並且可以動態改變網頁的內容、佈局和樣式。JavaScript 是網頁互動性和動態效果的主要實現方式之一。

總的來說，HTML 用於定義網頁的結構和內容，CSS 用於設計網頁的樣式，JavaScript 用於實現網頁的互動性和動態效果。這三種語言在網頁開發中都起著重要的作用，需要熟練掌握。

Q: 請給一個html範例

only_html.html

這個範例中，我們使用了幾個常見的 HTML 標記：

<!DOCTYPE html> 宣告了這個文件是一個 HTML 文件。
<html> 元素定義了整個 HTML 文檔的開始和結束。
<head> 元素包含了關於 HTML 文檔的元信息，如標題、描述等等。
<title> 元素定義了網頁標題，顯示在瀏覽器的標題欄中。
<body> 元素定義了網頁的主要內容，包括文本、圖像、表格等等。
<h1> 元素定義了一個大標題，這裡是 "歡迎來到我的網站！"。
<p> 元素定義了一個段落，這裡是一段簡短的介紹。
<img> 元素用於插入圖像，src 屬性指定了圖片的 URL，alt 屬性定義了圖片的替代文本，用於在圖片無法顯示時提供替代信息。
這個範例中的 HTML 標記比較簡單，但是它們展示了 HTML 標記的用法和基本結構。

Q: 請給一個CSS範例，且檔名需要是css

styles.css

在這個樣式表中，我們定義了幾個常用的樣式：

標題樣式：h1、h2、h3 元素使用相同的字體，字體加粗，字體大小分別為 36px、24px、18px，並且在下方留出 10px 的空白。
段落樣式：p 元素使用相同的字體，字體大小為 16px，行高為 1.5 倍字體大小，並且在下方留出 20px 的空白。
超連結樣式：a 元素設置顏色為藍色，並且去掉下劃線。當滑鼠懸停在連結上時，添加下劃線。
圖片樣式：img 元素設置最大寬度為 100%，高度自適應，並且在下方留出 20px 的空白。同時添加一個 1px 的 #ccc 實線邊框。
這樣，當我們在 HTML 中引入這個樣式表時，就可以使用這些樣式，而不必在每個 HTML 文件中都定義一次。
